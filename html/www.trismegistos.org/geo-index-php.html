The Places database, expanding the geographic database of the Fayum project, is a set of currently 14,197 places in (and some outside) Egypt. It contains the currently 113,927 toponyms attested in texts from Egypt (BC 800 - AD 800), but is also linked to the provenance field in Trismegistos Texts
