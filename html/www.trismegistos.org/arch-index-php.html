The Archives database, built on the Leuven Homepage of Papyrus Archives, is a set of currently 464 collections of texts in antiquity, mainly in Egypt, and the 16,461 texts that are part of these archives. It is searchable separately, leading to the texts themselves

description
The People database, building on the Prosopographia Ptolemaica, is a complex set of prosopographical and onomastic databases. It currently contains 458,045 attestations of personal names of non-royal individuals living in Egypt between BC 800 and AD 800, including all languages and scripts and written on any surface.
