description
BODO: Bibel+Orient Datenbank Online. The BIBLE+ORIENT Museum Database is a web based database created for the electronic management of ancient Near Eastern image and object data. The numerous data fields enable not only a precise cataloging of the museum inventory but also systematic searches for purposes of scientific research. Furthermore, the database admits the consultation of digitalized catalogs of the collections, the management of image series (e. g. slide shows), the making available of dynamic e-Learning content and the fast creation of PowerPoint presentations as well as publications. The user has at his disposal images in an unlimited amount of views for each object, each in the following four formats: thumbnail, normal size, PowerPoint format and print resolution. The idea of an internet database allows not only world wide access but also stimulates the cooperation between partners who want to share the data with one another. The database was designed so that image and object data from an unlimited number of institutions and private persons can be managed. Currently, we are working on national, European and global levels to include into our database objects from other iconographically relevant collections as well.
