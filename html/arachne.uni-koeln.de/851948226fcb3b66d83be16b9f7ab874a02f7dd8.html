Arachne Monument Browser

description
Arachne Monument Browser. Usually object information in the database Arachne is developed by inquiries. With this in mind, browsing is the common navigation form for information retrieval. Although Arachne already provides this opportunity within the attached context browser, it can be realized more explicitly within some exemplified, figuratively created monuments and types. Ara Pacis Sarcophagi Pergamon Altar Trajan’s Column Mercator’s Map of Cologne.
