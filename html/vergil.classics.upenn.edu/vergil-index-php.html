description
The Vergil project: Resources for Students, Teachers, and Readers of Vergil. The Vergil Project is a resource for students, teachers, and readers of Vergil's Aeneid. It offers an on-line hypertext linked to interpretive materials of various kinds. These include basic information about grammar, syntax, and diction; several commentaries; an apparatus criticus; help with scansion; and other resources.
