description
The Comprehensive Aramaic Lexicon. The CAL is a text base of the Aramaic texts in all dialects from the earliest (9th Century BCE) through the 13th Century CE, currently with a database of approximately 2. 5 million lexically parsed words, and an associated set of electronic tools for analyzing and manipulating the data, whose ultimate goal is the creation of a complete lexicon of the language. IT IS A WORK IN PROGRESS, not a completed dictionary. Accordingly, any citations for scholarly purposes should include the date when the data was found.
