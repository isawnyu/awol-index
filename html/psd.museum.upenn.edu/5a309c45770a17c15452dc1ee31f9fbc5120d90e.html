description
The Pennsylvania Sumerian Dictionary. Welcome to the website of the Pennsylvania Sumerian Dictionary Project (PSD). The PSD is preparing an exhaustive dictionary of the Sumerian language which aims to be useful to non-specialists as well as Sumerologists. In addition, we are developing tools and datasets for working with the Sumerian language and its text-corpora. All materials will be made freely available on this website.
