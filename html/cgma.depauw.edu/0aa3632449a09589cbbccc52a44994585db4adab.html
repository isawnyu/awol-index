description
MAGIS: Mediterranean Archaeology GIS. <blockquote class="tr_bq">Welcome to MAGIS, an inventory of regional survey projects in the greater Mediterranean region. To search the database, please choose one of the options at the left. <br/><br/>The <a href="http://cgma. depauw. edu/MAGIS/Spatial/spatial. php"> </a>  allows users to zoom to a specific region of the Mediterranean or to view all projects currently entered (for which we have coordinates). <br/><br/> <br/> The <a href="http://cgma. depauw. edu/MAGIS/MAGIS_GE_RT/CGMA_MAGIS. kmz"> </a>  downloads an automatically updating. kmz file that, when opened in <em>Google™ Earth 5. 0</em> or later, allows users to browse and interact with all survey projects in the MAGIS database for which spatial (coordinate) data are available. <a href="http://cgma. depauw. edu/MAGIS/Help/help. php#ge_browse">More info. </a><br/>  <br/> The <a href="http://cgma. depauw. edu/MAGIS/DB_Search/search_main. php?clear_form=Y"> </a>  allows queries based on specific fields in the database. <br/><br/> We also invite scholars to enter the metadata about their own projects by using the <a href="http://cgma. depauw. edu/MAGIS/DB_Admin/db_submit/register. php"> </a>  form. <br/><br/> Our <a href="http://cgma. depauw. edu/MAGIS/Help/help. php"> </a>  pages offer detailed explanations about the search and data entry options, as well as browser support and the user interface. You can access the Help pages directly on each page by clicking on the symbol. <br/>. If you have <em>corrections to</em> or <em>comments about</em> projects already in the system, please contact us at   (a link to this address exists at the bottom of every page). Contact us at the same address if you are interested in the code developed for MAGIS. Thank you!. </blockquote>.
